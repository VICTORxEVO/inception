Compute

Redesigned Compute Unit

Each RDNA 4 Compute Unit houses 64 shader processors arranged across two SIMD32 units, retaining the RDNA 3 layout while dramatically improving IPC through wider issue ports and a deeper out-of-order instruction scheduler.

⚙️

Dual SIMD32 Design

Two SIMD32 vectors per CU enable 64 FP32 operations per clock, with a unified 32-bit / 16-bit execution path for optimal throughput.

🔁

Wider Issue Width

The scheduler now issues up to 4 independent instructions simultaneously per SIMD, reducing stalls and improving occupancy.

💾

Larger Register File

Expanded vector and scalar register files reduce spilling to local memory, boosting throughput for complex shader workloads.

⚡

Improved Branch Execution

Enhanced scalar branch predictor and converged execution paths cut divergence penalties for game shaders and compute kernels.

🧵

Async Compute

8 hardware compute queues per shader engine allow full overlap of graphics and compute workloads for FSR and VRS operations.

🏎️

Clock Headroom

TSMC 4 nm enables sustained boost clocks above 3.0 GHz on flagship dies, a significant step up from RDNA 3's 2.7 GHz target.

Ray Tracing

3rd‑Gen Ray Accelerators

RDNA 4 introduces third-generation hardware Ray Accelerators — one per CU — delivering 2× the ray-box and ray-triangle intersection throughput versus RDNA 3.

RT

Dedicated Intersection Engine

Each Ray Accelerator can process two BVH (Bounding Volume Hierarchy) traversal steps per clock — double compared to RDNA 3 — enabling complex scenes with thousands of light bounces at playable frame rates.

The improved RA also handles procedural intersection shaders entirely in hardware, offloading work from the shader processors.

AI Acceleration

New AI Matrix Cores

Dedicated WMMA (Wave Matrix Multiply Accumulate) units have been added to every CU, providing up to 4× the AI inference throughput of RDNA 3.

AI

WMMA Matrix Units

Two WMMA units per CU operate on 16×16 matrix tiles in FP16, BF16, INT8, and INT4 formats. These units are entirely independent of the shader processors, so AI inference runs in parallel with rendering.

This powers FidelityFX Super Resolution 4 (FSR 4), AMD's first machine-learning upscaler, delivering native-quality visuals at a fraction of the rendering cost.

FidelityFX Super Resolution 4

Machine-learning upscaling now leverages the WMMA cores, producing temporally stable, near-native image quality from low-resolution inputs.

🎬

AV1 Encode / Decode

Dual AV1 encode engines offer up to 8K60 encoding, enabling high-quality streaming and recording with minimal performance impact.

🧠

ROCm & DirectML

Full ROCm 6 support and optimised DirectML kernels allow RDNA 4 to run LLM inference and diffusion models at competitive performance.

Memory & Cache

Faster Memory Subsystem

RDNA 4 pairs GDDR6 memory with a redesigned Infinity Cache hierarchy and a wider internal data fabric to keep the shader engines fed.

📦

256-bit GDDR6

A 256-bit memory bus with 20 Gbps GDDR6 modules delivers up to 640 GB/s of raw bandwidth on the RX 9070 XT.

🗄️

Infinity Cache 2.0

64 MB on-die cache with improved hit-rate prediction and wider internal bandwidth reduces effective memory latency by up to 35%.

🔗

PCIe 5.0 ×16

Native PCIe 5.0 support doubles slot bandwidth to 128 GB/s, eliminating any CPU-to-GPU transfer bottleneck for data-heavy workloads.

🖥️

DisplayPort 2.1a

Up to 80 Gbps UHBR20 signalling supports 8K144 or 4K360 over a single cable, alongside HDMI 2.1b for 10K output.

RDNA Lineage

From RDNA 1 to RDNA 4

Four generations of architectural evolution, each delivering a meaningful leap in performance-per-watt.

RDNA 1 — 2019 (7 nm)

Introduced the Compute Unit redesign with dual SIMD32 execution, replacing GCN's SIMD16. Launched with the RX 5700 series.

RDNA 2 — 2020 (7 nm)

Added hardware ray tracing accelerators and Infinity Cache. The RX 6000 series competed directly with Nvidia's Ampere lineup.

RDNA 3 — 2022 (5 nm + 6 nm)

Chiplet multi-die design debuted. Doubled AI throughput with WMMA and introduced DisplayPort 2.1. RX 7000 series.

RDNA 4 — 2025 (4 nm)

Monolithic die on TSMC 4 nm. 3rd-gen ray accelerators (2× RT throughput), new WMMA cores (4× AI), and FSR 4. RX 9000 series.
